## Where the CS Capstone Gets Real

The CS Capstone with Open Source SLU puts students on production software teams building for actual organizations. Three tracks: client-driven product development, entrepreneurship, and research.

<details>
<summary>Highlights</summary>

- Build production software on a structured team with a graduate student Tech Lead
- Industry-standard workflows: sprint planning, code review, version control, and testing
- Automatically join our paid developer bench for ongoing work opportunities
- Professional portfolio of GitHub repositories and deployed applications

</details>
