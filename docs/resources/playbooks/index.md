# Playbooks for our Programs

Each playbook provides operational guidance, curriculum resources, and instructional materials for running Open Source with SLU programs.

## [CS Capstone](https://github.com/oss-slu/course_capstone)

Course content and GitHub Project issues for the undergraduate CS Capstone experiential learning course.

## [Developing OSS Products](https://github.com/oss-slu/course_dossp)

Course material and automation system for DOSSP, the graduate-level course where Tech Leads develop skills in technical leadership, product stewardship, and open source software development.

## [BOLT Workshop](https://github.com/oss-slu/workshop_bolt)

Resources for the Building Open Leadership Toolsets workshop, an intensive onboarding experience for open source leaders and maintainers.
