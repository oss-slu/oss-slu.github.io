# Guides for Participants

Each guide provides role-specific orientation, expectations, and resources for participants in Open Source with SLU programs.

## [Team Members](https://github.com/oss-slu/handbook_developer)

The Developer Handbook covers onboarding, development workflows, team collaboration practices, and technical standards for students contributing to open source products.

## [Tech Leads](https://github.com/oss-slu/handbook_tech_lead)

The Tech Lead Handbook covers product stewardship, team coordination, code review practices, and the leadership responsibilities of graduate students serving as Tech Leads.

## [Industry Fellows](https://github.com/oss-slu/handbook_industry_fellows)

The Industry Fellows Handbook covers mentorship practices, event expectations, team matching, and how Fellows engage with student teams throughout the academic year.
