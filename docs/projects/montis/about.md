## Overview

Montis Peaks 3D is an interactive mountaineering platform that enables climbers, trekkers, and researchers to explore 3D mountain terrain digitally before expeditions. It combines photorealistic 3D terrain visualization with intelligent route planning that considers terrain slope and avalanche/rockfall hazards. Users can create and save climbing routes, annotate hazard zones, and analyze terrain features through slope, risk, and segmentation overlays.

> **Note:** This project had a Fall 2025 team only and is currently stable with no active team.
